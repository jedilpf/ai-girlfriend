## ⚠️ 注意事项

1. **先搭 Edge Function 再写前端**，否则 API 调不通
2. **环境变量不要提交到 Git**，用 `.env.local`
3. **Vercel 部署前**先在本地 `vercel dev` 测试
4. **IndexedDB 按角色隔离**，切换角色时清空当前消息列表
