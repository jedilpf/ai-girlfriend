## 2026-04-17 12:33 团队评审反馈

**评审人指出三个问题：**

### 问题 1：Web Speech API iOS 兼容性
- **风险**：Safari 支持一般，iOS 上可能不可用
- **解决方案**：
  - 优先：引导用户使用输入法自带的语音键盘（零开发，全平台可用）
  - 备选：Whisper API 转写
  - Web Speech API 降级为备选（仅 Chrome/Android）
- **状态**：✅ 已修改

### 问题 2：好感度变化判断逻辑不明确
- **风险**："夸她+3~5" 怎么实现？靠 LLM 还是关键词？
- **解决方案**：**用 LLM 判断，不额外花钱**
  - System Prompt 要求 LLM 回复末尾附加 `[affection:+1]` 或 `[affection:+3]` 等标记
  - 前端解析标记后更新好感度
  - 一次 API 调用同时完成对话和好感度判断
- **状态**：✅ 已修改

### 问题 3：ElevenLabs TTS 免费额度限制
- **风险**：免费版 10000 字符/月，聊天频繁可能不够
- **解决方案**：
  - 改自动播放为**手动点喇叭图标播放**（大幅降低消耗）
  - 10000 字符/月 ≈ 每天 300+ 条回复，Demo 阶段足够
  - 备选：浏览器 SpeechSynthesis API（免费但声音机械）
- **状态**：✅ 已修改

## 2026-04-17 12:35 团队深度评审（13 个问题）

### 🔴 致命漏洞

**1. API Key 前端暴露**
- **问题**：纯前端项目，Key 写在前端 = F12 就能看到，盗用额度
- **解决方案**：Vercel Edge Function 代理层，Key 存环境变量，前端永远接触不到
- **状态**：✅ 已解决（新增 `api/chat/index.ts`）

**2. CORS 问题**
- **问题**：通义 DashScope 和 ElevenLabs 不允许浏览器直接跨域调用
- **解决方案**：同上，Edge Function 代理，浏览器只和自己的域名通信
- **状态**：✅ 已解决

**3. iOS 语音输入不可用**
- **问题**：Web Speech API 在 Safari/iOS 支持很差
- **解决方案**：优先用输入法自带麦克风键盘，零开发全平台可用
- **状态**：✅ 已在 v1.1 解决

**4. PWA 推送没后端做不了**
- **问题**：Service Worker Push API 必须有服务器
- **解决方案**：Demo 阶段模拟推送（打开页面时收到），完整版需要后端
- **状态**：✅ 已标注限制

### 🟡 架构隐患

**5. localStorage 容量有限**
- **问题**：5MB 上限，长期聊天容易撑满
- **解决方案**：聊天记录改用 IndexedDB（无限制），localStorage 只存配置
- **状态**：✅ 已修改

**6. 换设备 = 清零**
- **问题**：没有云端同步，清缓存/换手机数据丢失
- **解决方案**：Demo 阶段首次使用时提示用户，v2.0 加用户登录+云端同步
- **状态**：✅ 已标注

**7. ElevenLabs 免费额度**
- **问题**：10000 字符/月，200 次对话耗尽
- **解决方案**：改自动播放为手动点喇叭，每天 300+ 条回复够用
- **状态**：✅ 已在 v1.1 解决

**8. 对话历史没有角色隔离**
- **问题**：所有角色共用历史，切换角色后 LLM 会 OOC
- **解决方案**：按角色 ID 分开存储对话历史
- **状态**：✅ 已修改

### 🟢 设计层面

**9. 好感度变化规则不明确**
- **解决方案**：LLM 回复末尾返回 `[affection:+N]` 标记，一次调用搞定
- **状态**：✅ 已在 v1.1 解决

**10. 没有内容安全机制**
- **解决方案**：Edge Function 层加关键词过滤，命中返回安全回复
- **状态**：✅ 已新增

**11. 没有加载/错误状态**
- **解决方案**：添加 loading 动画（"正在输入..."）+ 失败重试 + 超时提示
- **状态**：✅ 已新增

**12. 长按交互不具体**
- **解决方案**：改为双击，iOS/Android 行为一致
- **状态**：✅ 已修改

**13. 成本估算偏乐观**
- **解决方案**：修正成本表，实际 ~70-100 元/月（重度使用），Demo 远低于此
- **状态**：✅ 已修正
